# 📚 Book Recommender System

This project is a Flask web application that recommends books to users based on collaborative filtering.  
It uses pre-computed similarity scores and book popularity data to suggest relevant books.  
The app is interactive, simple, and can be deployed on Render or Heroku for free.

## 📂 Files Included

- `app.py` : Flask app for frontend and recommendation logic  
- `templates/` : HTML templates for UI  
- `static/` : CSS and image files for styling  
- `popular.pkl`, `pt.pkl`, `books.pkl`, `similarity.pkl` : Preprocessed model and data files  
- `requirements.txt` : Python dependencies  

## 🚀 Features

- Popular books display  
- Search-based book recommendations  
- Responsive UI using HTML/CSS  
- Ready for deployment  

## 🛠 Technologies Used

- Python, Flask  
- Pandas, NumPy, scikit-learn  
- HTML, CSS  
- Pickle (for saving model data)
